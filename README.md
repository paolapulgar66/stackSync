# StackSync - Integración Completa con Docker, GraphQL, Axios y MongoDB

Este proyecto es una aplicación full-stack automatizada mediante contenedores **Docker** que demuestra el funcionamiento real e integrado de una arquitectura cliente-servidor moderna.

## 🚀 Tecnologías Utilizadas e Integración
- **Frontend (Cliente):** React + Vite. Utiliza **Axios** para enviar peticiones estructuradas hacia el servidor.
- **Backend (API):** Node.js + **GraphQL** (esquemas, queries y mutations) para procesar la lógica de negocio y autenticación.
- **Base de Datos:** **MongoDB** para la persistencia y almacenamiento de datos en tiempo real.
- **Orquestación:** **Docker Compose** para unificar, aislar y levantar todo el ecosistema con un solo comando.
